MAIL MERGE: The Mail Merge feature of MS Word can be used to create and send documents containing similar information to many people. Some content in the document is unique for each recipient. The document is typed once. The unique information in each document comes from a list containing the names of the recipients and other required details, called the data source. Mail Merge combines the document and the data source to create a personalized copy for each recipient.

Components of Mail Merge:

1. Main document: The main document contains the text and graphics to be sent to all the people.

2. Data Source: The data source is a file that contains the information to be merged with the main document.

3. Merged document: In the main document, we need to identify the information that we wish to change or personalize for each recipient. Each details is represented by a Merge field in the letter.
